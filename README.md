# CoderConnect

## how to install
```
cd frontend
yarn install

cd .. 
cd backend 
pip install -r requirements.txt
yarn install
pip install flask-cors
```

## Starting mySQL
```
run Command Prompt as admin
net start MySQL80
```

## how to run
```
cd frontend 
yarn dev

cd .. 
cd backend
python app.py
```

## Installing Dummy Data
```
//make sure you are in backend folder
cd backend
python insert_dummy_data.py
```
